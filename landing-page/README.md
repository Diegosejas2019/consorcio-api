# GestionAr Landing Page

Landing estatica basada en el diseno recibido en `C:\Users\Diego\Downloads\test (1)`.

## Uso

```bash
npm run dev
```

Luego abrir:

```text
http://localhost:4173
```

Tambien se puede abrir `index.html` directamente en el navegador, porque no requiere build ni dependencias.

## Estructura

- `index.html`: landing completa con HTML, CSS y JavaScript embebidos.
- `public/uploads/`: assets originales incluidos con el diseno.
- `server.mjs`: servidor estatico minimo para previsualizar la landing.
